Introduction

mvMotionAI is an AI library module (currently a static library, hopefully in version 00-02-00 and above as a DLL & shared library).

C/C++ steering behaviour library (AI) with Lua scripting for games and simulations.

Compiles under the following development environments

  • Ubuntu
    • Code Blocks with GCC compiler
  • Windows XP
    • Code Blocks with GCC MinGW compiler
    • MSVC++6
    • MS Visual Studio 2005

This library is a general purpose steering behaviours library for games and simulations that is easily pluggable into different applications.

How to integrate mvMotionAI library into your application

The library is coded in C++ with interfaces in both C++ and the scripting language Lua.

First created in January 2006, then the source code was released as open source in June 2006 under the MIT License.

The library was started as a reusable code for applications (such as games), focusing on the area of artificial intelligence known as steering behaviours or motion/movement based such as path-following and seeking & evading another body.

Motion based behaviours with scripting functionality.
