  1. Overview

    2. Basic motion/velocity integrator library for 3D worlds with basic behaviours for C++ programs. An important part is the scripting capabilities which allows easy file loading of AI settings. Currently the script library Lua is used and is open source under the MIT license.

    mvMotionAI is a C/C++ & Lua based artificial intelligence module dealing with motion type actions such as one body follow another or moving towards a single point. These types of actions are referred as steering behaviours. As well as these common behaviours, this library uses a scripting language, so the various settings of each behaviour can be changed without recompiling the program.

    mvMotionAI uses a OpenGL function interface, as most of the functions in the library have a prefix of 'mv'. The main class in the library is the mvWorld, which contains all of the minor classes. mvWorlds contain mvForce (gravity, wind, direction pull), mvObstacles (non-moving objects which are not yet implemented), mvWaypoints (points or markers in the world).

    In the library, each moving item such as a car, a bird or person is represented as a mvBody. Each mvBody has various properties such as its position, maximum speed, direction in the world. These bodies can be grouped together, Each behaviour has already been written in the library, so there is no feature yet implemented which allows completely new behaviours to created at run-time.

    There are four implemented behaviour in the library which are:

    MV_SEEKmoves towards a way point
    MV_FLEEmoves away from a way point
    MV_CLONEcopies the velocity of another body
    MV_PURSUITchases after a moving body
    MV_SIMPLE_FLOCKa group flocking behaviour consisting of three types of behaviours which is similar to boids. They are attraction, separation and cohesion.

    The scripting language in this case is Lua. By combination of behaviours above you can create a different motion. In the test application, the Lua script (Test.lua) contains some behaviours which the bodies in a worlds follow.

    For example, to build a world containing a body at (4,1,0) moving towards a way point at (10,11,4), the following Lua code is:

    world1 = mvAddWorld('HELLO_WORLD');
    body1 = mvAddBody(world1,'MV_PARTICLE','MV_AABOX',4,1,0);
    waypoint1 = mvAddWaypoint(world1,'MV_SINGLE_WAYPOINT','MV_AABOX',10,11,4);
    -- comments start with --
    -- this
    mvSetDefaultWaypointForBody(world1,waypoint1,body1);
    mvAddBehaviourToBody(world1,body1,'MV_SEEK');
